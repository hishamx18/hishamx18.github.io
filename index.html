Library Asssistant Robot

Hover here for more details

Design and development of a smart, autonomous robot to assist library users in locating books, managing inventory, and enhancing the overall library experience.
The robot, with the support of Microcontrollers (Arduino), Single Board Computers (Raspberry Pi), RFID technologies and Robot Operating System (ROS), interacts with users and navigates through the library environment.
It also utilizes LIDAR sensors integrated with ROS to simultaneously localize and map to follow obstacle avoidance algorithms.
