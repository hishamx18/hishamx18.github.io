A-star Algorithm

A-star path planning algorithm implemented in a custom Graphical User Interface using Python.

Details

A* is implemented on a 2D grid-based environment where users can interactively set up the planning scenario. The GUI (Graphical User Interface) allows the user to d`efine the start and goal positions by clicking on the desired cells and create obstacles manually by clicking and marking grid cells as occupied. The obstacles act as barriers that the planner must navigate around.

The A* algorithm will then search for a feasible path from the start to the goal, taking into account the user-placed obstacles. It dynamically updates and displays the explored nodes and the final planned path directly within the GUI, providing an intuitive visualization of the search process.


Skills learnt

Python, Path planning algorithms, Data structures, Problem solving and debugging
